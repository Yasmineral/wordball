## Setup

- run "npm install"
- run "npm run dev" to start server
- run "npm test" to run tests

## Tech Stack
 
- code written in javascript (ES6) 
- backend server: Express.js 
- database: MongoDB
- frontend: jQuery(currently) and React
- api to check for words: http://www.anagramica.com/api
